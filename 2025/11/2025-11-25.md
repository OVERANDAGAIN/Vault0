# **🎨 PPT 页面内容（可直接粘贴到幻灯片）**

**Motivation & Background**

* Many cooperative MARL tasks are **partially observable**
  → each agent only sees local observations.

* Training usually follows **CTDE**
  (Centralized Training, Decentralized Execution).
  → but policies still rely on **limited local information** during execution.

* In complex environments (MPE, LBF, RWARE):

  * Observations are **incomplete**
  * Rewards are often **sparse**
  * Agents must coordinate **precisely and simultaneously**

* Existing MARL methods struggle because:

  * Policies lack **global state understanding**
  * Representation learning (LIAM / SIDE) is **decoupled** from policy learning
    → leads to *distribution mismatch*
  * Exploration methods (curiosity, novelty search, EMC, EOI) are not designed for **multi-agent partial observability**

---

# **🚀 Core Motivation**

> **Can agents learn a meaningful “belief” of the unobserved global state, using only their own observation?**
> **And use this belief to guide cooperative behavior and multi-agent exploration?**

大家好，首先我们来看一下这篇论文所面对的核心背景。

在很多协作型多智能体任务中，比如 MPE、Level-Based Foraging、Robot Warehouse，环境都是**部分可观测的**：每个智能体只能看到非常有限的局部信息。而在 CTDE 框架下，虽然训练时可以依赖全局信息，但**执行时每个 agent 依然只能使用自己的局部观察**，这会对协作造成极大的困难。

并且这些环境通常奖励是**稀疏**的，而且需要**严格的协作 timing**。例如 LBF 中多个 agent 必须同时吃掉一个 food，RWARE 中多个机器人必须协同搬运任务。这对探索和状态理解提出了更高要求。

现有方法为什么难以解决？原因有三个：

第一，policy 直接输入局部观测，很难恢复全局状态的隐含结构；
第二，已有的表示学习方法，比如 LIAM、SIDE，它们的 latent embedding 与策略优化是**脱耦的**，容易出现 distribution mismatch；
第三，基于 curiosity 或 novelty search 的探索方法大多是为单智能体设计的，在部分可观测的多智能体环境中效果不佳。

因此，本文提出的核心问题是：

我们能不能让每个智能体只依赖自己的局部观察，就能**推断出一种对全局状态的 latent belief 表示**？
并进一步利用这种 belief 去**提升协作、改善探索**？

这就是 SMPE² 想解决的根本动机。
