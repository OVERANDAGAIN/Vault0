# ✅ **（1）SMPE² 状态建模 + ELBO 目标（公式 2）**

```
maximize_{ω_i, φ_i, ψ_i}  V_SM(ω, ψ) + λ · ELBO(ω_i, φ_i ; o_i, w_i · o_{-i})
```

---

# ✅ **（2）ELBO（Evidence Lower Bound）**

```
ELBO(ω_i, φ_i ; o_i, w_i · o_{-i})
= E_{z_i ~ q_{ω_i}} [ log p_{φ_i}( w_i · o_{-i} | z_i ) ]
  - KL( q_{ω_i}(z_i | o_i) || p(z_i) )

p(z_i) = N(0, I)
```

---

# ✅ **（3）重构损失（Self-supervised Reconstruction Loss, 公式 3）**

```
L_rec(ω_i, φ_i, φ_i^w)
= || w_i^e · o_{-i}  -  w_i · ȯ_{-i} ||^2
```

（其中 ȯ_{-i} = decoder 输出）

---

# ✅ **（4）稀疏正则（防止 w 全为 0，公式 4）**

```
L_norm(φ_i^w) = - || w_i ||_2^2
```

---

# ✅ **（5）KL 正则（使 z_i 为 variational belief，公式 5）**

```
L_KL(ω_i) = KL( q_{ω_i}(z_i | o_i) || p(z_i) )
p(z_i) = N(0, I)
```

---

# ✅ **（6）两个 critic 的 TD 损失（公式 6）**

### Central Critic

```
L_critic(ξ)
= ( r_t^i + V_{ξ'}^π(s_{t+1}) - V_{ξ}^π(s_t) )^2
```

### Filtered Critic（核心创新）

```
L_critic^w(φ_i^w, k)
= ( r_t^i + V_{k'}^π(ŝ_{t+1}) - V_k^π(ŝ_t) )^2

where   ŝ = o_i ⊕ ( w_i · o_{-i} )
```

---

# ✅ **（7）Encoding 总损失（公式 7）**

```
L_encodings
= L_critic^w
  + λ_rec · L_rec
  + λ_norm · L_norm
  + λ_KL · L_KL
```

---

# ✅ **（8）Actor Loss（带 entropy 正则 + advantage）**

```
L_actor(ψ_i)
= - β_H · H( π_{ψ_i}(a_t^i | h_t^i, z_t^i) )
  - log π_{ψ_i}(a_t^i | h_t^i, z_t^i)
      · ( r_t^i + V_{ξ'}^π(s_{t+1}) - V_{ξ}^π(s_t) )
```

---

# ✅ **（9）最终 SMPE 总损失**

```
L_SMPE = L_actor + L_critic + L_encodings
```
