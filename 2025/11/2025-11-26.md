# **Part 1 — 问题背景 & Motivation（3–4 页）**

### **1.1 多智能体协作任务的困难点**

* 全合作 MAS（MPE/LBF/RWARE）
* 部分可观测（Dec-POMDP）
* CTDE：训练集中、执行分散
* 典型困难：

  * **局部观测导致信息缺失**
  * **多智能体之间缺乏共享状态**
  * **稀疏奖励（尤其 LBF & RWARE）**
  * **探索难：需要精准协作、长期规划**

**一句话引出问题：**

> 现有 MARL 方法在部分可观测 + 稀疏奖励的协作任务中表现极差。

---

### **1.2 现有方法的不足（非常重要）**

* 纯 RL：信息不足、策略不稳定
* 基于 ICM/curiosity/novelty 的探索策略：对部分可观测任务适应差
* LIAM/SIDE 等 representation learning 方法：

  * 重构任务与 RL 无耦合 → **表示与策略不一致（distribution mismatch）**
  * 表示往往没有策略意义

**引出本文核心动机：**

> 能否学习一个“对策略真正有用”的 belief 表征，使 agent 能在部分可观测环境中恢复关键隐变量，并促进协作探索？

# **Part 2 — 方法直觉：SM + AM（3–4 页）**

### **2.1 State Modelling（SM）直觉**

* 从自己的观测 $o^i$ 中推断一个 latent belief $z^i$
* $z^i$ 代表：“我认为全局状态可能是什么样子”
* 用 VAE 方法，让 $z^i$ 不仅可重构，也稳定可采样
* $z^i$ 用于：

  * actor 输入
  * intrinsic reward（novelty via SimHash）
  * TD critic

**直观图（可画）：**
o^i → encoder → z^i → critic & actor & exploration

---

### **2.2 Agent Modelling（AM）直觉**

* 我无法直接看到别人
  → 但我能推断别人可能看到了什么（reconstruction）
* 为避免无用特征干扰
  → 用 filter $w^i$ 从 $o^{-i}$ 中筛选有用维度
* 最终只保留：

  * 对策略有用
  * 对可推断性的特征

**一句话总结：**

> AM 通过可微软滤波器 $w^i$ 自动选择“有价值”特征。

---

### **2.3 SM + AM 在一起构成核心思想**

* SM：构建 belief
* AM：确保 belief 的输入是可推断且策略相关
* value critic 把两者耦合起来

**一句话：**

> SM 负责 “表示全局状态”，AM 负责 “选择哪些特征值得表示”。

### **3.1 Encoder–Decoder（ED）模块**

* 输入：自己观测 $o^i$
* 输出：对别人观测的重构 $\hat{o}^{-i}$
* 加 KL：让表示稳定
* 加 filter $w_i^e$：只重构被选中的维度

讲清楚 $w^i$ 和 $w_i^e$ 区别（观众一定会问）：

| 名称      | 功能        | 来源          | 用途                |
| ------- | --------- | ----------- | ----------------- |
| $w^i$   | 为 RL 选择特征 | 从 TD loss 学 | 构建 filtered state |
| $w_i^e$ | 为重构选择特征   | 从 L_rec 学   | 指定 decoder 的目标    |

---

### **3.2 AM Filters 模块**

* 用 sigmoid gating 选择有用维度
* 稀疏化 $L_{norm}$
* 价值 guided $L^w_{critic}$
* 重构 guided $L_{rec}$

**关键句**：

> AM filter 是通过“自监督（可推断性）+ RL（策略价值）”共同训练出来的软选择器。

---

### **3.3 双 critic 结构**

* Central critic：MAA2C 的 $V_\xi(s)$
* Filtered critic：$V_k(\hat{s})$

重要点：

* 过滤后的状态 $\hat{s}$ = “我相信的别人状态”
* filtered critic 是将 representation 直接与策略优化耦合的核心

---

### **3.4 Intrinsic reward (adversarial exploration)**

* 用 SimHash(z) 计算 novelty
* 探索奖励加到 extrinsic reward 上
* 提供新的轨迹 → 强化 ED → AM 也更新得更好
* 是一种 **对别人 adversarial 的探索**（扰动别人重构任务）

---

### **3.5 最终 loss（重点公式一页）**

* Actor loss
* Central critic loss
* Filter critic loss
* Reconstruction + KL + Norm
* 总 loss = 三大模块联合训练

# **Part 6 — Discussion（你需要的最后总结）（1–2 页）**

你可以从以下角度发散：

### **6.1 表达学习 + 强化学习的耦合是未来趋势**

SMPE² 属于表征学习/RL 一体化的强趋势。

### **6.2 局限性**

* representation 可解释性有限
* exploration 高维任务中可能不稳定
* decoder 训练成本偏高
* AM filter 选择机制仍偏“软”、“模糊”

### **6.3 未来方向**

* 用 transformer 构建更强的 encoder
* 基于 attention 替代软 gating
* 多模态 belief（可扩展到视觉）
* 不依赖 decoder 的 lighter state modelling

# **Part 7 — 总结（1 页）**

三个最核心 take-away：

1. **State Modelling（belief learning）让部分可观测 MARL 强大起来**
2. **Agent Modelling filters 解决“哪些特征值得学”这个关键问题**
3. **探索奖励与表示学习耦合，解决稀疏奖励任务**

一句话总结论文贡献：

> SMPE² 提供了一套系统化框架，将“状态建模 + 特征选择 + 对抗式探索”统一到一个可微结构中，在部分可观测的协作任务中显著超越现有方法。
