# Q1：Dreamer系列要使用 RSSM 作为世界模型？


| 主题          | 要点                                                          | 作用/意义                |
| ----------- | ----------------------------------------------------------- | -------------------- |
| RSSM 定义     | 循环状态空间模型（Recurrent State-Space Model），用于建模序列数据的时间依赖         | 更好刻画随时间变化的动态过程       |
| 与 RNN 的区别   | 在循环体内显式建模两类状态：随机状态 + 确定性状态                                  | 同时表达环境的随机性与确定性       |
| 优势 1：双性质建模  | 同时处理确定部分（如物理规律）和随机部分（如噪声干扰）                                 | 更贴近真实世界环境，建模更准确      |
| 优势 2：抽象表示学习 | 通过编码器把原始输入映射到抽象表示，关注控制任务关键特征                                | 提升学习效率与泛化能力          |
| 优势 3：高效采样   | 使用 factorized dynamics predictor 快速采样 imagined trajectories | 便于在“想象轨迹”里训练/规划，计算更省 |
| 优势 4：可解释性   | 观察状态表示可一定程度理解模型如何“理解环境”                                     | 帮助分析模型行为与决策依据        |
