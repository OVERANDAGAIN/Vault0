## 用户满意度 / 留存信号：Observable vs. Latent

### 1. 先区分两类信号

一次 episode log 可观测为：

$$\mathcal{E} = \{(y_t, u_t, ad_t, click_t, topic_t, done_t)\}_{t=1}^{T}$$

这些是 **observable signals**：

$$O_t = \{T, done_t, ad_t, click_t, topic_t, u_t\}$$

但用户满意、反感、疲劳等不是直接观测量，而是 **latent user states**：

$$Z_t = \{satisfaction_t,\ annoyance_t,\ fatigue_t\}$$

只能估计：

$$\hat{Z}_t = f_\theta(\mathcal{E}_{1:t})$$

---

### 2. Reward 可以使用 latent estimator

Reward 是优化目标，本身允许引入估计量：

$$r_t = R_{\text{ads}}(t) + \lambda_s \hat{s}_t - \lambda_q \hat{q}_t - \lambda_f \hat{f}_t$$

其中：

* $\hat{s}_t$：estimated satisfaction
* $\hat{q}_t$：estimated annoyance
* $\hat{f}_t$：estimated fatigue

这些可以由：

* LLM judge
* BERT / DeBERTa classifier
* human-labeled reward model

估计。

---

### 3. Metrics 应优先使用 observable signals

Metrics 用于方法独立评价，因此主指标应尽量来自可观测行为：

$$M_{\text{primary}} = g(\mathcal{E})$$

例如：

$$\text{Retention} = \frac{T}{T_{\max}}$$

$$\text{CAR} = \frac{\sum_t click_t}{\sum_t ad_t}$$

$$\text{AdLoad} = \frac{\sum_t ad_t}{T}$$

$$\text{TSR} = \frac{\sum_{t=2}^{T}\mathbb{I}(topic_t \neq topic_{t-1})}{T-1}$$

---

### 4. Latent estimates 只作为 secondary metrics / diagnostics

模型估计的满意度、反感度可以保留，但不作为 primary metric：

$$M_{\text{secondary}} = \{\hat{s}_t, \hat{q}_t, \hat{f}_t\}$$

作用：

* 解释 failure case
* 分析用户心理状态
* 对 reward estimator 做 sanity check

---

### 5. 如果有真人实验，latent states 可通过 survey 观测

真人实验中，可以通过 Likert scale 近似观测 latent variables：

$$s^{\text{human}} \in \{1,2,3,4,5\}$$

例如：

* 任务是否被解决？
* 广告是否自然？
* 是否愿意继续使用？
* 是否感到被打扰？

这可以作为：

$$Z_t^{\text{gold}}$$

用于验证：

$$\hat{Z}_t \approx Z_t^{\text{gold}}$$

---

## 当前设计结论

$$\textbf{Reward:} \quad \text{can use latent estimates}$$

$$\textbf{Metrics:} \quad \text{should prioritize observable behavior}$$

$$\textbf{Human study:} \quad \text{can validate latent user states}$$

最终采用：

| 层级                | 信号来源                       | 用途              |
| ----------------- | -------------------------- | --------------- |
| Primary Metrics   | observable episode log     | 方法独立评价          |
| Secondary Metrics | LLM / classifier estimates | 诊断分析            |
| Human Evaluation  | user survey / rating       | gold validation |
