## 制作U盘时，写保护无法格式化
win32diskimage会给U盘加上写保护，最终导致无法格式化
### 使用Rufus重新制作
在格式化之后：
- [x] 不要使用视频中的软件，使用Rufus

### 重新格式化方法
U盘无法格式化因为写有保护的这样做
win + r 
cmd
输入diskpart
     list disk
     select disk 对应磁盘编号
     输入clean
     输入creat partition primary
     输入active
     输入format fs=fat32 quick（搬自csdn）
就可以格式化了，随后一楼说用rufus，具体教程csdn上也有。
然后更改时区那点，要先联网，然后根据一楼和弹幕的提示输入相关指令就可以了
