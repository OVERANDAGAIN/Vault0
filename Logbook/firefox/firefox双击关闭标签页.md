# Answers

来源：[如何开启 Firefox 双击关闭标签页功能[Firefox]](https://www.zhihu.com/question/364363515)[^1]

首先在地址栏上输入「
```
about:config
```


」，然后再在页面中的搜索栏输入「

```
browser.tabs.closeTabByDblclick
```

」。接着双击它，当看到「值」那里变成「true」就可以了。

很简单，很方便，现在你已经可以直接双击这个标签页来关闭它了。
