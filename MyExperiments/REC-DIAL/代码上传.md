## 方案 A：先修 SSH，

你第一次 SSH 报的是：

```text
Host key verification failed.
```

这是因为 GitHub 主机指纹没有加入 `known_hosts`。你当时应该输入 `yes`，但没有成功。

现在执行：

```bash
ssh-keyscan github.com >> ~/.ssh/known_hosts
```

然后测试：

```bash
ssh -T git@github.com
```

如果看到类似：

```text
Hi OVERANDAGAIN! You've successfully authenticated...
```

说明 SSH 通了。

然后把 remote 改回 SSH：

```bash
git remote set-url origin git@github.com:OVERANDAGAIN/diag.git
git push -u origin feature/formulation
```
