src/recdiag/core/types.py

---

### 2. Monitor

已改：

```text
src/recdiag/core/monitor.py
```

新主接口是：

```python
build_state(profile, history, episode_log) -> PlannerState
```

不再走旧的：

```python
evaluate_turn(...) -> signals
```

当前有两个版本：

```text
RuleBasedMonitor   # baseline
LLMBasedMonitor    # 同样输出 PlannerState，失败时 fallback 到 rule
```

这样后期换 LLM Monitor 不需要再改 Env / Planner / Reward。

---

### 3. Planner 和 ActionSpace

已改：

```text
src/recdiag/core/planner.py
```

现在 action space 是 formulation 里的混排一维动作：

```text
[-1, ad_0, ad_1, ..., topic_0, topic_1, ...]
```

新增：

```text
ActionSpace
BasePlanner
RandomPlanner
RuleBasedPlanner
LLMPlanner
```

Planner 现在只做一件事：

```python
act(state, action_space) -> PlannerAction
```

### 4. Speaker

已改：

```text
src/recdiag/core/speaker.py
```

新接口：

```python
generate_utterance(state, action, topic_pool) -> AgentUtterance
```

Speaker 只负责实现 Planner action，不重新决定是否插广告或切 topic。

src/recdiag/core/simulator_state.py
src/recdiag/core/simulators/profile_simulator.py

---

### 6. DialogueEnv

已改：

```text
src/recdiag/env/dialogue_env.py
```

旧接口：

```python
step(bot_utterance)
```

已经改成 formulation 接口：

```python
reset(initial_query=None) -> PlannerState
step(action: PlannerAction) -> next_state, reward, done, info
```

现在 `DialogueEnv.step()` 内部完整执行：

```text
state
→ action
→ speaker response
→ user raw feedback
→ ad event
→ next_state
→ reward
→ trajectory step
```

### 7. Reward

新增：

```text
src/recdiag/reward.py
```

实现：

```text
task reward
satisfaction reward
continuation reward
raw ad reward
terminal reward
```

广告 reward 按你确认的方式处理：

```text
只要 action 是广告动作，就直接使用广告原始 reward_value
不根据 clicked / accepted 再加复杂系数
```

---

### 8. Metrics

新增：

```text
src/recdiag/metrics.py
```

Metrics 只依赖：

```text
EpisodeLog
AdEvent
exit_reason
```

不依赖 reward breakdown。

已实现：

```text
TaskSuccessRate
EarlyFailureRate
Revenue
RevenueDensity
AdExposureRate
ParetoVector
```

这和你 metrics 文件里的独立评价原则一致：metrics 不用自定义 reward 评价自己。

---

### 9. Runner

已改：

```text
scripts/run_simulation.py
```

现在主循环变成：

```python
state = env.reset()

while not done:
    action = planner.act(state, env.action_space)
    next_state, reward, done, info = env.step(action)
    planner.update(state, action, reward, next_state, done)
    state = next_state
```

`DialogueAgent` 不再作为主流程使用。
