单个 planner、单个 seed 的 PowerShell 指令如下。

## 1. 进入项目目录并设置环境变量

```powershell
cd D:\REC-DIAL\recdiag
$env:PYTHONPATH = "src"
$env:PYTHONUNBUFFERED = "1"
```

## 2. 跑单个实验

例如跑 `greedy_ad`，seed 为 `7`，3 个 episodes：

```powershell
C:\Users\Administrator\AppData\Local\Programs\Python\Python313\python.exe scripts\run_simulation.py --config config.json --num-episodes 3 --monitor llm --planner greedy_ad --speaker llm --reward-judge llm --user-response hybrid --seed 7
```



## 5. 可选 planner 列表

```text
noad

rule
greedy_ad
conservative_ad
llm
```



# 1. 正式实验配置

建议先跑：

```text
7 planners × 3 seeds × 3 episodes = 63 episodes
```

planners：

```text
noad
random
random_ad
rule
greedy_ad
conservative_ad
llm
```

seeds：

```text
7, 13, 21
```

每个 run：

```text
--num-episodes 3
```

# 2. 多窗口并行指令

## 窗口 1：`noad`, `random`

```powershell
cd D:\REC-DIAL\recdiag
$env:PYTHONPATH = "src"
$env:PYTHONUNBUFFERED = "1"

$planners = @("noad", "random")
$seeds = @(7, 13, 21)

foreach ($planner in $planners) {
  foreach ($seed in $seeds) {
    Write-Host "=== Running planner=$planner seed=$seed ==="
    C:\Users\Administrator\AppData\Local\Programs\Python\Python313\python.exe scripts\run_simulation.py --config config.json --num-episodes 3 --monitor llm --planner $planner --speaker llm --reward-judge llm --user-response hybrid --seed $seed
  }
}
```
