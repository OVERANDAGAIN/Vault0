# Solutions
1. 安装CUDA10.2 ( python3.7.11 )（可以在应用-程序里卸载旧的cuda相关程序)，以及 `C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA` 可以卸载后在手动删除残余文件夹
2. 安装cudnn,解压后把文件拷贝到cuda目录
3. 安装anaconda,pycharm,python等。
4. 虚拟环境创建，安装torch，测试
